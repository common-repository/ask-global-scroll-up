== Description ==

### Ask-Global-scroll-up by http://ask-global.com


This plugin will add scroll to top button in your wordpress website. 


== Installation ==

This plugin is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. upload Ask-Global-scroll-up folder to wp-Content/plugins folder.

2. Activate the plugin from Dashboard / Plugins window.
